#NFT TEMACH

El Temach es un youtuber mexicano cuya comunidad utiliza el símbolo de la cadena militar como un signo de conexión, pertenencia y motivación para el Modo Guerra.

Este proyecto es el lanzamiento de 400 NFT (cadenas virtuales) para crear una comunidad de compas dentro de la blockchain, donde todo es fijo y permanente. Incluso próximamente se podrá colaborar con El Temach para desarrollar proyectos usando estas cadenas, que funcionarán a modo de identificación de los compas más firmes.

Las cadenas se pueden mintear (solo una por compa) de forma gratuita (a excepción del gas de la transacción, que no supera los 5 céntimos), pero hay una comisión del 1% en la compraventa de las cadenas. Según en qué momento se mintee la cadena, se obtendrán diferentes tipos:

La Suprema (la primera que se mintea)
La Dorada (las 20 primeras)
La Roja (aparece cada 10 cadenas)
La Silver (el resto)
La secreta (?)
